## 1. エグゼクティブサマリー

### プロジェクト概要

YONA Beach Club（プーケット）と同等規模（全長約48m、幅18m、500名収容、22mインフィニティプール搭載）のフローティング・ビーチクラブをハワイで展開する事業の実現可能性調査。

### 結論サマリー

| 項目 | 結論 |
|------|------|
| **技術的実現可能性** | 可能（ただし96トンプールの自由表面効果が最大の設計課題） |
| **法的実現可能性** | 可能（「固定プラットフォーム＋米国建造テンダー」方式でPVSA準拠） |
| **市場性** | 極めて高い（ハワイに独立型ビーチクラブ/デイクラブが**皆無**＝市場ギャップ） |
| **総投資額** | **$34.5M〜$87.5M**（約52億〜131億円） |
| **年間売上見込** | **$38M〜$56M**（保守的〜基本シナリオ） |
| **EBITDA見込** | **$14M〜$33M**（マージン37〜59%） |
| **投資回収期間** | **1.5〜3.5年**（基本シナリオ） |
| **最大のリスク** | 環境訴訟（Hawaii Superferry型の事態）、許認可の長期化 |
| **推奨所在地** | **コオリナ（Ko Olina）が第一候補** |
| **プロジェクト開始〜開業** | **5〜7年** |

### 推奨建造方式

**海外建造バージ（非推進プラットフォーム）＋ 米国建造テンダー艦隊 = 約$18〜22M**

これが最もコスト効率が高く、PVSA（旅客船サービス法）を完全に回避できる最適な方式。

### 3.5 プール工学（最大の技術課題）

96トンの水は**自由表面効果**により船の復原力を著しく低下させる。自由表面効果の補正量はプール幅の**3乗**に比例。

**対策オプション**:
1. **バッフル（仕切り板）設置**: プール内分割で実効幅を削減
2. **プール分割**: 複数の小プールに分割
3. **海水循環式**: 船体一体化構造（衛生基準との整合が課題）
4. **能動的バラスト制御**: プール水の影響をリアルタイム相殺
5. **船体幅拡大**: カタマラン/トリマラン構造が有利

【推定】USCGは96トンプール水を含む復原性計算を極めて厳格に審査する。設計初期段階から海事エンジニアリング会社との協働が**不可欠**。

## 4. 海事規制・許認可マトリクス

### 4.1 PVSA（旅客船サービス法）— 最重要法規制

**法的要件**: 米国の2地点間の旅客輸送は、米国建造・米国船籍・米国市民所有・米国人乗組員の船舶のみ可能。

**違反罰則**: 旅客1名あたり**$798〜$996** → 500名で1航海あたり**$399,000〜$498,000**の罰金

#### 推奨法的戦略:「固定プラットフォーム＋米国建造テンダー艦隊」

| 要素 | 内容 |
|------|------|
| **本体（バージ）** | 海外建造可能。沖合に恒常的に係留。推進システムを撤去/最小化し「固定海上施設」として分類 |
| **テンダー艦隊** | **必ず米国建造**。旅客を港から本体施設まで輸送（PVSA完全準拠） |
| **法的手続き** | CBPへの**Binding Ruling Request**提出が最初のアクション |

**MARAD免除（Waiver）の見込み**: 【推定】**極めて困難（成功率10〜15%以下）**。米国造船業界の組織的反対が最大の障壁。

### 4.2 USCG Subchapter K（46 CFR Part 114-122）

500名旅客・100GT以上の船舶として**明確にSubchapter K適用範囲**。固定施設分類の場合でも、USCGは同等の安全基準を課す可能性が極めて高い。

| 要件カテゴリ | 主な内容 |
|-------------|---------|
| 復原性 | 非損傷時・損傷時基準適合、旅客集中・風圧計算、**プール自由表面効果計算** |
| 消防安全 | A級仕切り（60分耐火）、スプリンクラー必須、固定式消火設備 |
| 救命設備 | 全員分の救命胴衣＋25%予備、救命いかだ（全員収容）、EPIRB |
| ADA | エレベーター/リフト、バリアフリー経路、障害者用トイレ |
| 避難分析 | 全乗船者が30分以内に救命設備到達の証明 |

**COI（Certificate of Inspection）取得**: 図面審査6〜12ヶ月 → 建造監督12〜24ヶ月 → 最終検査1〜3ヶ月

## 6. 環境規制・コミュニティ戦略

### 6.1 環境影響評価（EIS）— 絶対に回避してはならない

**Hawaii Superferry事件（2007〜2009年）の教訓**:

| 経緯 | 結果 |
|------|------|
| EISをスキップしFONSIで済ませた | 環境団体（Sierra Club）が訴訟 |
| ハワイ最高裁がEIS実施命令 | 運航停止 |
| 州議会が特例法（Act 2）で運航許可 | 最高裁がAct 2を**違憲**と判決 |
| **$3.5億投資の事業が完全崩壊** | 会社破産 |

**→ EISは最初から計画に組み込む。回避の試みは致命的。**

### 6.2 環境規制の主要ポイント

| 規制 | 管轄 | フローティングクラブへの影響 |
|------|------|------------------------|
| **HEPA（Hawaii Environmental Policy Act）** | OEQC | EA → EIS必須（保全地区・州水域利用） |
| **NEPA** | USACE（Lead Agency） | 連邦環境レビュー必須 |
| **クジラサンクチュアリ** | NOAA/DLNR | 12〜5月に厳しい制約（冬季運営縮小が現実的） |
| **サンゴ礁保護** | DLNR-DAR | アンカー設置はサンディボトムのみ |
| **NDZ（No Discharge Zone）** | EPA | **ハワイ全水域で汚水排出完全禁止** |
| **MMPA** | NOAA | 海洋哺乳類へのハラスメント禁止（水中騒音含む） |
